============
Installation
============


Stable release
--------------

To install gds_tidy3d, run this command in your terminal:

.. code-block:: console

    $ pip install -e .[dev]

This is the preferred method to install gds_tidy3d, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for gds_tidy3d can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/mustafacc/gds_tidy3d

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/mustafacc/gds_tidy3d/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/mustafacc/gds_tidy3d
.. _tarball: https://github.com/mustafacc/gds_tidy3d/tarball/master
